# A tool made to apply my tweaks with ease.

## Quick Start

### Visual Studio 2022

1. **Open** `DARKSTARtool.sln` in Visual Studio 2022
2. **Right-click** project → **Publish**
3. **Configure**: Folder → Self-contained → win-x64 → Single file
4. **Click** Publish
5. **Find** your exe: `bin\Release\net8.0-windows\win-x64\publish\DARKSTARtool.exe`

## How to use
Place NSudo.exe in the same folder and run DARKSTARtool.exe as **Administrator**<br>
